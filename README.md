Smart Personal Data Organizer (SPDO)
📌 Overview

The Smart Personal Data Organizer (SPDO) is a Python-based console application designed to help users efficiently manage personal information such as contacts, notes, and tasks in one place.

This project demonstrates the use of basic Python programming concepts like functions, lists, dictionaries, and user input handling through a menu-driven interface.

🚀 Features
📇 Contact Management
Add new contacts (Name, Phone, Email)
View all contacts
Search contacts by name

📝 Notes Management
Add personal notes
View saved notes

✅ Task Management
Add tasks with deadlines
View all tasks

🛠️ Tech Stack
Language: Python 3

Concepts Used:
Lists & Dictionaries
Functions
Loops & Conditionals
User Input Handling

Application Menu
1. Add Contact
2. View Contacts
3. Add Note
4. View Notes
5. Add Task
6. View Tasks
7. Search Contact
8. Exit
